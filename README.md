ToDoApp

1) git clone https://github.com/ammarlow/ToDoApp.git
2) run composer install
3) make sure .env is correct, otherwise run php artisan key:generate
4) create database named 'todolist'
5) run php artisan migrate
6) done!

API:

1) Run postman laravel.test/api/todos
2) Get the json data
3) Run npm install
4) Run npm run dev
5) goto laravel.test/apiTodo to see the API

*laravel.test = url()
